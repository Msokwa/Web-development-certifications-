This training program consisted of two intensive 14-week courses focused on front-end development.

1. HTML and CSS
Instructor/Organization: ReDI School of Digital Integration.

Duration: 52 hours (16/03/2023 to 23/06/2023).

Core Competencies:

Fundamental web development structure and editors.

CSS Selectors and typography (Web Fonts).

Responsive Web Design for multi-device compatibility.

Project Work: Successfully built three complete websites.

2. JavaScript
Instructor/Organization: ReDI School of Digital Integration.

Duration: 48 hours (13/09/2023 to 14/12/2023).

Core Competencies:

Core logic: Variables, Types, Operators, Conditionals, and Loops.

Functional Programming: Arrays, Functions, and Object-Oriented basics.

Dynamic Web Interaction: DOM manipulation and API integration.

Data Management: Handling JSON and external data sources.

🛠 Technical Skill Set
Markup & Styling: HTML5, CSS3, Responsive Design.

Scripting: Modern JavaScript (ES6+), DOM API.

Data: JSON, API Fetching.

Static Site Portfolio: Developed three websites focused on layout and design principles using HTML/CSS.
